# Projects

This repository showcases a collection of systems-focused and performance-oriented projects spanning C++ engine architecture, GPU programming, and applied deep learning.

The work emphasizes low-level system design, parallel computation, and scalable data pipelines, with a focus on real-time performance and efficient resource management.

## GPU Animation Engine (C++ / DirectX 11 / HLSL)
A real-time 3D animation engine that offloads skeletal animation and hierarchy computations entirely to the GPU using compute shaders. Demonstrates parallel processing, GPU resource management, and engine-level architecture, including multi-stage compute pipelines and structured buffer design.


## Image Captioning System (Python / TensorFlow)
A multimodal deep learning pipeline combining CNN encoders and attention-based sequence decoders to generate image captions. Focuses on model training, optimization, and generalization, addressing challenges such as vanishing gradients and overfitting.


Across these projects, the repository highlights:

Systems engineering: modular architecture, memory/resource management, and pipeline orchestration
Parallel and high-performance computing: GPU compute shaders, multistage dispatch, and data-parallel workloads
Machine learning fundamentals: deep learning architectures, training workflows, and model optimization
